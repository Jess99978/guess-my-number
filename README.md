## 功能

- 隨機生成一個數字讓玩家猜測
- 提示玩家猜測的結果（太高或太低，並且驗證是否輸入要求範圍內的數字）
- 計算玩家猜測的次數及最高分數
- 顯示三種不同的遊戲結果
- 重置及退出遊戲

## 安裝與執行
1. clone 這個儲存庫：
   ```bash
   git clone https://github.com/Jess99978/guess-my-number.git
   ```
2. 打開 `index.html` 來運行遊戲
3. 或是直接從 [這裡](https://jess99978.github.io/guess-my-number/) 開始遊戲
